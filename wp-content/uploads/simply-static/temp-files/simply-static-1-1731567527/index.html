Frequently Asked Questions

Berikut adalah beberapa pertanyaan umum
yang ditanyakan orang tentang layanan ini.

Apakah lembaga keuangan perlu menyiapkan server untuk dapat menggunakan layanan dots Core?

Seluruh layanan dots Core berbasis Web sehingga memberikan fleksibilitas bagi lembaga keuangan untuk mengatur arsitektur instalasi sistem yang akan digunakan. Saat ini umumnya terdapat beberapa skema yang dapat dipertimbangkan, yakni diantaranya:

3. Cloud based, yakni menggunakan layanan dengan cloud server yang dimiliki oleh PT Dimensi Kreasi Nusantara sehingga tidak perlu menyediakan perangkat server. Opsi ini umum dilakukan oleh Koperasi Simpan Pinjam yang baru berdiri atau memiliki aset yang belum cukup besar

1. On-premise, yakni aplikasi di-install pada perangkat server milik sendiri dan diletakan pada jaringan kantor BPR / Koperasi. Opsi ini umum dilakukan pada BPR dengan mempertimbangkan POJK dan SEOJK terkait Penyelenggaraan Teknologi Informasi pada BPR

2. Co-location, yakni aplikasi di-insall pada perangkat server milik sendiri namun fisik server diletakan pada Data Center. Saat ini kami bekerja sama dengan beberapa Data Center rekanan di Jakarta Duren Tiga dan Bogor untuk layanan colocation server. Opsi ini umum dilakukan pada BPR dengan mempertimbangkan POJK dan SEOJK terkait Penyelenggaraan Teknologi Informasi pada BPR

Apakah layanan aplikasi Core System yang disediakan termasuk dengan layanan migrasi data dari sistem lama?

Ya, kami juga akan turut membantu lembaga keuangan untuk melakukan migrasi data dari sistem lama / pembukuan manual dengan Excel ke dots Core System

Berapa jumlah maksimum nasabah / rekening yang dapat dioperasikan pada sistem?

Tidak ada batasan jumlah rekening, jumlah nasabah, jumlah nasabah, maupun jumlah user yang dilimatasi pada sistem.

Apakah memungkinkan untuk meintegrasikan dots Mobile Platform atau dots Connect API ke aplikasi CBS saya yang berjalan?

Silahkan hubungi kami untuk assessment dan penjajakan terlebih dahulu atas kebutuhan tersebut.

Apakah dots CBS mendukung multi-cabang dan terkonsolidasi secara real-time?

Ya, sistem dots CBS mendukung operasional Kantor Cabang (serta Kantor Cabang Pembantu untuk KSP) secara real-time, terintegrasi, dan terkonsolidasi sehingga memungkinkan adanya transaksi antar kantor. Konektivitas antar kantor cabang tidak membutuhkan ISP khusus dan dapat dicapai dengan layananÂ dots VPN IP.

Apakah sistem telah mendukung pelaporan eksternal ke OJK? Apabila di kemudian hari terdapat perubahan atas hal tersebut apakah akan terdapat pembaharuan sistem?

Sistem dots Core BPR telah terintegrasi dengan pelaporan eksternal ke OJK (Laporan Bulanan / Lapbul, SLIK, OBOX), PPATK, LPS, dll secaraÂ seamless. Untuk setiap perubahan regulasi yang dirilis oleh OJK, maka sistem akan secara otomatis mendapatkan pembaruan untuk mendukung perubahan regulasi tersebut.
